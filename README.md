## 角色分工

> 本项目所有角色由同一人承担。

| 角色 | 职责 |
|------|------|
| **论文阅读者** | 精读论文，提炼核心贡献与技术细节 |
| **文献检索者** | 检索相关文献，梳理研究脉络 |
| **Review 撰写者** | 撰写论文 Review 正文 |
| **反思记录者** | 记录个人启发与学习心得 |
| **项目管理** | 管理 Git 版本历史、维护文件结构 |

| 阶段 | 时间 | 任务 |
